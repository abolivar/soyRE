# Glossary

| Español | Inglés técnico | Nota |
|---|---|---|
| Inmueble | `property` | Entidad central. |
| Propietario | `owner` | Dueño del inmueble. |
| Consignación / mandato | `mandate` | Autorización comercial. |
| Publicación | `listing` | Ficha o anuncio. |
| Visita | `showing` | Actividad comercial. |
| Oferta | `offer` | Oferta de compra/alquiler. |
| Operación | `deal` | Transacción comercial. |
| Alquiler | `lease` | Contrato de alquiler. |
| Inquilino | `lessee` | Evitar `tenant`. |
| Organización SaaS | `organization` | Cliente SaaS. |
| Comisión | `commission` | Pago/split comercial. |
| Auditoría | `audit-log` | Registro de acciones. |
