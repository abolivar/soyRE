# Codex Code Review Prompt

Revisa este cambio como si fueras tech lead del proyecto.

Verifica:

- Cumplimiento de alcance.
- Criterios de aceptación.
- Seguridad.
- Multi-tenancy.
- Permisos.
- Tests.
- Cambios fuera de alcance.
- Consistencia con la arquitectura.
- Documentación actualizada.

Entrega:

- Hallazgos críticos.
- Hallazgos menores.
- Riesgos.
- Recomendación: aprobar / solicitar cambios.
