# Audit Architecture

## Objetivo

Registrar acciones críticas sobre propiedades, documentos, mandatos, deals, comisiones y permisos.

## Acciones futuras críticas

- Crear, editar o archivar una propiedad.
- Cambiar precio.
- Aprobar publicación.
- Subir, aprobar o rechazar documentos.
- Aceptar oferta.
- Cerrar deal.
- Aprobar comisión.
- Marcar comisión como pagada.
- Cambiar permisos.
