# Real Estate Stack Bootstrap Pack

Este paquete contiene:

- `stack-bootstrap-basics.md`: decisiones básicas del stack.
- `codex-bootstrap-stack-prompt.md`: prompt listo para Codex.
- `repo-bootstrap-template/`: archivos base sugeridos para el repositorio.

Uso recomendado:

1. Crear un repo vacío.
2. Copiar el contenido de `repo-bootstrap-template/` al repo.
3. Reemplazar `docs/product/foundational.md` con el documento fundacional completo.
4. Pegar `codex-bootstrap-stack-prompt.md` en Codex como Tarea 0.
