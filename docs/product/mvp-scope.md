# MVP Scope

## Dentro del MVP

- Auth básica propiedad del API.
- Organizaciones.
- Usuarios, roles y memberships.
- Propiedades.
- Propietarios.
- Mandatos.
- Documentos básicos.
- Workflows básicos.
- Listings internos.
- Visitas.
- Ofertas.
- Deals básicos.
- Comisiones básicas.
- Dashboard operativo.
- Auditoría básica.

## Fuera del MVP

- App móvil nativa.
- Marketplace público.
- MLS completo.
- Property management enterprise.
- Contabilidad completa.
- IA avanzada.
- Integraciones complejas con todos los portales.
- Firma electrónica avanzada.
